#  CS3750 Alpha Version  Functional Requirements

## Overview
This Alpha demonstrates two major functional points of the RV Park Management System:

1. **Reservation Management** – create, edit, and cancel reservations, process payments and refunds.
2. **Occupancy Reports**  daily site vacancy and occupancy data retrieved live from the database.

## Database Interaction
The UI communicates with the Prisma database layer to:
- Read and write reservation data  
- Update availability  
- Generate reports dynamically  

## Purpose
This document summarizes what functionality is included in the Alpha release per assignment guidelines.
